### Установка (Windows, npm/pnpm)

1. Локально
```
npm install --save f-logger --latest
```

или

```
pnpm install --save f-logger --latest
```

2. Глобально
```
npm install --global --save f-logger --latest
```

или

```
pnpm install --global --save f-logger --latest
```
